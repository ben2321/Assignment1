Project Idea

Overview:

The idea for a project could be to create a To-Do application where the user can create list of things to do for their day/ week/ month and possibly year, the user could also use the app for recipes and shopping lists. The lists can be saved through the apps system along with the function to create new lists. The To-Do list app would be able to send notifications to the user when the end date for the list is close.

Description:

The reason why the To-Do list would be a useful app to have is because it will help users keep track of all the things that are needed to be done through the notification system, but it will also help with the smaller less important things needed to be done. This would be a helpful thing for people who lead very busy lives to keep track of all the things they need to do, and it can help for cooking by storing recipes. The idea would be interesting in the way that it could be created, experimenting with different features and including being able to edit the To-Do list to make it look good with different fonts, colours of the font and background etc. In the modern world there are so many things that are needed to be remembered in a day-to-day life so the To-Do list app with inbuilt notifications would be very helpful. The app will begin with a blank main screen where in the top right corner will have an addition symbol (+) where you can add your first To-Do list, the To-Do list creation would start with dot points where the user would write what they need to do, if the list is one thing then the submit button would be on the bottom of the screen and this would save the list and take you back to the main screen and your To-Do list would be there. If the list was longer than one step, then the user would write the first step and then press enter which will drop to the next step with a new dot point. To edit the To-Do list the user would need to go into a saved list and press the options button at the top right which would be represented with three dots stacked on each other, in this edit tab would be where the user can edit the fonts, the colours of the text and background and the notification/ alarm function for reminding the user about the To-Do list.

Tools/Tech:

I would not exactly know the exact hardware or software needed to create the app, but I know that there would need to be code written so that the actions can be completed properly by the user, there will also be a need for somewhat modern computer hardware for the processing and graphical editing.

Skills Required:

The skills needed would not be hard to find/ learn, there would be many articles and tutorials on how to create a simple To-Do list but the more advanced parts of the To-Do list like the notification function would be rather difficult to a person who is just beggining in app development. The prototype used to test the layout could be created by using websites like proto.io or anyother website or software that allows you to create a prototype app.
